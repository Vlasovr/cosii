# Отчет по лабораторной работе 2 (КИХ `Р` с прямоугольным окном + однополюсный НЧ)

## Цель
Спроектировать и проверить работу однородного, КИХ и БИХ фильтров на сигнале из ЛР1.

## Исходные данные и искажения
- Базовый сигнал: сгенерирован с теми же параметрами, что и в ЛР1.
- Искажения:
  - синусоидальная помеха `700 Гц`;
  - белый шум.

## Реализованные фильтры
1. Однородный фильтр:
   - КИХ скользящего среднего с окном `M = 9`.
2. КИХ фильтр (`Р`, прямоугольное окно):
   - режекторный (полосо-заграждающий) фильтр;
   - полоса подавления: `650..750 Гц`;
   - число коэффициентов: `101`.
3. БИХ фильтр:
   - однополюсный НЧ-фильтр;
   - частота среза: `450 Гц`.

## Основные формулы
- Ядро однородного фильтра:
  - `h[n] = 1/M, n=0..M-1`
- Идеальная импульсная характеристика режекторного КИХ:
  - `h[0] = 1 - (w2 - w1)/pi` в центральной точке
  - `h[k] = - (sin(w2*k) - sin(w1*k)) / (pi*k), k != 0`
- Однополюсный НЧ:
  - `alpha = exp(-2*pi*fc/fs)`
  - `y[n] = (1-alpha)*x[n] + alpha*y[n-1]`
