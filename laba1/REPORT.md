# Отчет по лабораторной работе 1 (вариант DIF БПФ)

## Цель
Реализовать основные операции цифровой обработки сигналов для двух периодических сигналов и проверить корректность в ручной и библиотечной реализациях.

## Исходные данные
- Частота дискретизации: `2048 Гц`
- Длительность: `2.0 с`
- Сигналы: периодические комбинации синусоид.

## Реализованные операции
1. Линейная свертка:
   - Ручная: прямое суммирование во временной области.
   - Библиотечная: `numpy.convolve`.
2. Полная корреляция:
   - Ручная: свертка с развернутым по времени вторым сигналом.
   - Библиотечная: `numpy.correlate(..., mode="full")`.
3. Преобразование Фурье:
   - Ручное: БПФ DIF и обратное БПФ DIF.
   - Библиотечное: `numpy.fft.fft`, `numpy.fft.ifft`.
4. Проверка теоремы Фурье для свертки/корреляции.
5. Бенчмарк по нескольким значениям `N`.

## Основные формулы
- Бабочка DIF БПФ:
  - `a[k] = x[k] + x[k + N/2]`
  - `a[k + N/2] = (x[k] - x[k + N/2]) * W_N^k`
- Обратное БПФ:
  - `IFFT(X) = conj(FFT(conj(X))) / N`
- Теорема свертки:
  - `x * y = IFFT(FFT(x) * FFT(y))`
