# Лабораторные работы по ЦОС (ЛР1 + ЛР2)

В репозитории реализованы обе лабораторные работы в полном объеме.

## Что реализовано

### ЛР1 (вариант: БПФ с прореживанием по частоте, DIF)
- Генерация 2 периодических WAV-файлов (по 2.0 секунды).
- Ручная линейная свертка и ручная полная корреляция.
- Ручная БПФ DIF и ручное обратное преобразование.
- Библиотечные версии через NumPy (`fft`, `ifft`, `convolve`, `correlate`).
- Графики амплитудного и фазового спектров для `x`, `y` и результата свертки.
- Графики сигналов во временной области для `x`, `y`, свертки и корреляции.
- Проверка свертки/корреляции через теорему Фурье.
- Бенчмарк для разных `N` с графиком эффективности.

### ЛР2 (вариант: `Р` с прямоугольным окном, однополюсный НЧ-фильтр)
- Использование базового сигнала из ЛР1 и внесение контролируемых искажений.
- Однородный фильтр (скользящее среднее).
- КИХ режекторный фильтр, спроектированный методом прямоугольного окна.
- БИХ однополюсный НЧ-фильтр.
- Расчет коэффициентов и сохранение сводки.
- Построение АЧХ фильтров.
- Построение графиков во времени: вход, искаженный сигнал и выходы после 3 фильтров.

## Установка

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск

Запуск обеих лабораторных:

```bash
python3 run_all.py
```

Запуск только ЛР1:

```bash
python3 -m laba1.main
```

Запуск только ЛР2:

```bash
python3 -m laba2.main
```

## Папки с результатами
- `laba1/output`
- `laba2/output`

В каждой папке находятся WAV-файлы, графики и `сводка_результатов.txt` с численными результатами.
